# Credits & attributions

## OpenMoji
The colorful emoji icons across Discovery Quest games (world/station icons, lesson
illustrations, picture-match vocabulary, answer reveals) are **OpenMoji** — the open-source
emoji library.

- Source: https://openmoji.org
- License: **CC BY-SA 4.0** (https://creativecommons.org/licenses/by-sa/4.0/)
- The SVGs are used **unmodified**, rendered via `<Emoji>`
  (`packages/engine-ui/src/Emoji.jsx`); the used subset is bundled into each app's
  `public/openmoji/` by `scripts/gen-emoji-assets.mjs`.

## Lucide
UI chrome icons (locks, controls, arrows) are **Lucide** (https://lucide.dev), ISC license.
